# 百萬潛買｜物件推薦頁產出流程

## 你只需要做 4 步驟

1. 開住商網站篩選好條件（區域／價格／類型）
2. `Ctrl+S` 或「另存新檔」取得 HTML 原始碼
3. 貼給 Claude，說「幫我產物件推薦頁」
4. Claude 產出 → 推 GitHub → 給你網址

---

## Claude 自動完成的事

- 解析物件資料（標題、地址、坪數、價格、車位、MRT…）
- 產生精美 RWD 物件推薦頁 HTML
- 自動設定 **10天後過期**，到期跳轉 `findingsweethomekh.com`
- 嵌入 QR code（連結電子名片頁）、LINE、電話
- 推上 GitHub Pages，給你可分享網址
